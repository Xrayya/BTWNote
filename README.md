# BTWNote

Fun project to challenge my front-end, banc-end, and data structure skill

## Project Description

This project is a note-taking app. The inspiration came from obsidian. This will be a web app.

## Features

Here are the features that I want to implement in this project:

- [ ] Create a note
- [ ] Edit a note
- [ ] Delete a note
- [ ] Search for a note
- [ ] Tag a note
- [ ] Link a note
- [ ] Multi-link note
- [ ] Note versioning
- [ ] Note history
- [ ] Graph view
- [ ] Dark mode
- [ ] Markdown support

## Tech Stack
- **Front-end**:
  - [React](https://react.dev/) ([Vite](https://vitejs.dev/) template)
  - [TypeScript](https://www.typescriptlang.org/)
  - [SWC](https://swc.rs/)
  - [Tailwind CSS](https://tailwindcss.com/)
  - [React Query](https://react-query.tanstack.com/)
  - [React Router](https://reactrouter.com/)
  - [React Hook Form](https://react-hook-form.com/)
  - more as I go...
- **Back-end**:
  - [Bun](https://bun.sh/)
  - [TypeScript](https://www.typescriptlang.org/)
  - [Hono](https://hono.dev/)
  - more as I go...
- **Database**:
  - [PostgreSQL](https://www.postgresql.org/)
